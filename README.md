# 🤖 AI Prompt Generator

AI Prompt Generator adalah website untuk membantu membuat prompt AI yang lebih baik.

## ✨ Features

- 📋 Copy Prompt
- 💡 Generate prompt dari ide pengguna
- 🤖 Pilih AI target
- 🚀 Improve Prompts
- 📝 Pilih tipe prompt
- 🔴⚫ Red & Black Theme

## 📂 Files

- index.html → Website structure
- style.css → Design & theme
- script.js → Main logic
- favicon.png → Website icon

## 🚀 Usage

1. Clone repository
2. Buka `index.html`
3. Gunakan Prompt Generator
